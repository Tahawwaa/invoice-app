# 🧾 invoice-app

A lightweight invoice generator built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## ✨ Features

- 📄 Create professional invoices instantly
- 🖨️ Print or export as PDF with one click

---


---

## 🚀 Getting Started

```bash
git clone https://github.com/Tahawwaa/invoice-app.git
cd invoice-app
```

Open `index.html` in your browser — that's it.
